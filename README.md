## 🧰 Tools Overview

A small collection of pragmatic desktop tools I maintain and use in my daily workflows.

### **Mystic Crypt UI**
Cross-platform Java desktop application for encryption, key management, password storage, and checksum verification.

- Tree-structured storage for passwords and secure notes
- Generate and manage key pairs (1024 / 2048 / 4096 bit)
- Verify downloads using checksum algorithms (MD5, SHA variants)
- Obfuscate and de-obfuscate text via a configurable mapping

👉 https://astrapi69.github.io/tools/mystic-crypt.html

---

### **MP3 Tagger UI**
Cross-platform desktop tool for editing MP3 metadata (ID3 tags), including titles, artists, albums, track numbers, and cover artwork.

- Edit ID3v1 and ID3v2 tags
- Add or replace album cover images
- Batch-edit multiple MP3 files

👉 https://astrapi69.github.io/tools/mp3-tagger.html

---

### **PDF-to-Text UI**
Lightweight desktop application for extracting clean plain text from PDF documents.

- Convert PDFs to plain text for search, research, and AI prompts
- Batch process large sets of PDFs
- Reduce layout noise (columns, spacing artifacts) where possible

👉 https://astrapi69.github.io/tools/pdf-to-text.html
